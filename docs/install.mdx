## üêß **For Ubuntu/Debian (Linux)**

1. **Update packages**

   ```bash
   sudo apt install postgis postgresql-$(pg_config --version | cut -d' ' -f2 | cut -d'.' -f1,2)-postgis-3
   ```

   ```bash

   ```

   ```
   CREATE EXTENSION
   ```

   ```bash
   psql -d postgres -c "SELECT PostGIS_Version();"
   ```
