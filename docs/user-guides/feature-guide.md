## Field Management

### Creating a New Field

1. Navigate to **Field Management** → **Add New Field**
2. Use the map to **draw field boundaries**:
   - Click points to create polygon
   - Double-click to complete
   - System auto-calculates area in hectares
3. Enter **field details**:
   - Field name
   - Soil type (optional)
   - Elevation (optional)
4. **Save field**

### Viewing Field Details

Each field has a dedicated detail page showing:

- Field map with boundary
- Current weather conditions
- Recent activities
- Active alerts
- Crop history

## Weather Intelligence

### Current Weather

View real-time weather conditions for each field:

- Temperature
- Humidity
- Rainfall (last 24 hours)
- Wind speed and direction

### 7-Day Forecast

Detailed forecasts include:

- Daily temperature range
- Precipitation probability
- Wind conditions
- Confidence indicators

### Weather Alerts

Automatic alerts for:

- **Frost warnings** - Temperature < 2°C
- **Heavy rain** - Rainfall > 50mm expected
- **Heat stress** - Temperature > 35°C
- **Optimal spraying conditions** - Low wind, no rain

## Activity Logging

### Adding an Activity

**Desktop/Tablet:**

1. Go to **Activities** → **Add New Activity**
2. Select **field** and **activity type**
3. Enter **details**:
   - Date
   - Input products and quantities (optional)
   - Labor hours and workers
   - Notes
4. **Add photos** (optional)
5. **Save activity**

**Mobile (Field Use):**

1. Tap **+ button**
2. Select field (GPS auto-detection)
3. Choose activity type from quick list
4. Take photos with camera
5. Save (works offline, syncs when online)

### Activity Types

- Land Preparation
- Planting
- Fertilizer Application
- Pesticide/Fungicide Spraying
- Weeding
- Irrigation
- Pruning
- Harvesting

## Alert System

### Alert Types

**Weather Alerts:**

- Frost warnings
- Heavy rain
- Heat extremes
- Drought conditions

**Disease Risk Alerts:**

- Black Sigatoka risk (Banana/Plantain)
- Coffee Leaf Rust risk
- Crop-specific disease forecasts

**Operational Alerts:**

- Irrigation advisories
- Optimal harvest windows
- Spray timing recommendations

### Alert Workflow

1. **Receive notification** (push, SMS, or email)
2. **Open alert details** to see:
   - Field affected
   - Risk level
   - Expected impact
   - Recommended actions
3. **Acknowledge alert** when reviewed
4. **Take action** and log response
5. **Resolve alert** when completed

## Financial Tracking

### Recording Costs

**Input Purchases:**

1. Go to **Financial** → **Add Cost**
2. Select cost type:
   - Fertilizer
   - Pesticide
   - Seeds
   - Labor
   - Equipment
3. Enter details:
   - Product name
   - Quantity and unit
   - Cost in XAF
   - Date
   - Assign to field(s)
4. Save

### Recording Revenue

**Harvest Sales:**

1. Go to **Financial** → **Add Sale**
2. Select field and crop
3. Enter:
   - Quantity harvested
   - Quality grade
   - Price per unit (XAF)
   - Buyer (optional)
   - Date
4. Save

### Profitability Views

**Per Field:**

- Total costs
- Total revenue
- Gross profit
- Profit margin %

**Per Season:**

- Season costs breakdown
- Harvest revenue
- ROI calculation
